Ballerina Basic Types

                            
Simple types: 
/

 Always plain data. Includes nil, boolean, int, float, decimal. 
/

 Sequence: 
/

 Always plain data. Includes string, xml. 
/

 Structural: 
/

 Plain data if members are array/tuple, map/record, table. 
/

 Behavioural: 
/

 Not plain data. Includes function, object, error, stream, typedesc, handle.
